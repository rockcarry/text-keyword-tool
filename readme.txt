+-------------------------+
 文本关键词提取和标注工具
+-------------------------+

环境搭建
--------
1. 安装 python3
2. pip install summa


提取关键词
----------
get_keywords.py 脚本可以对 csv 文件提取关键词，保存到 keywords.txt 文件中
stop_word_list.txt 文件用于配置停用词

注意 csv 文件的第三列是评论内容，本工具只对第三列的评论内容进行处理
因此请确保 csv 文件中第三列存放的是评论内容


标注关键词
----------
add_keywords.py 脚本可以对 csv 文件进行关键词标注，对每一列的评论内容，查找匹配的三个关键字，并增加一列到表格
生成新的 out.csv 文件
